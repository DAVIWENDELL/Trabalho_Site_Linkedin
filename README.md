# Trabalho_Site_Linkedin

Este repositório contém o desenvolvimento de uma tela de login e cadastro inspirada no design do LinkedIn, criada como um trabalho acadêmico.

## Descrição do Projeto

O projeto consiste em uma interface web que simula as páginas de login e, possivelmente, de cadastro de uma rede social similar ao LinkedIn. O objetivo principal foi aplicar conhecimentos de desenvolvimento web frontend (HTML e CSS) para replicar o layout e a experiência visual de uma plataforma conhecida. A presença de um arquivo `salve.php` sugere uma tentativa inicial ou um placeholder para funcionalidades de backend, como o processamento de dados de formulário.

## Funcionalidades (Interface)

*   Página de login com campos para e-mail/telefone e senha.
*   Opções de login social (Google).
*   Design responsivo (implícito pelo uso de `meta viewport`).

## Tecnologias Utilizadas

*   **HTML5:** Estrutura e conteúdo das páginas web.
*   **CSS3:** Estilização e layout, buscando replicar o visual do LinkedIn.
*   **PHP:** (Potencialmente) para processamento de formulários, indicado pelo arquivo `salve.php`.

## Como Visualizar o Projeto

Para visualizar este projeto localmente, siga os passos abaixo:

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/DAVIWENDELL/Trabalho_Site_Linkedin.git
    cd Trabalho_Site_Linkedin
    ```
2.  **Abra o arquivo `index.html`:**
    Simplesmente abra o arquivo `index.html` em seu navegador web preferido. Não é necessário um servidor web para visualizar a interface estática.

    *Nota: A funcionalidade do `salve.php` exigiria um ambiente de servidor PHP (como Apache ou Nginx com PHP-FPM) para ser executada.*
